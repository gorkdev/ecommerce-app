# Admin — Next.js Dashboard

Admin dashboard for managing the store.

- **Stack:** Next.js 16 (App Router) · React 19 · TanStack Query · Tailwind CSS v4
- **Responsibilities:** dashboard & analytics, product/category CRUD with image upload,
  order management, coupon management, review moderation, user list.

Scaffolding lands in milestone **M9** (see root README roadmap).
